03 IA multimodal

VLM para validación contextual de imágenes

Diseño y entrenamiento de un modelo visión-lenguaje especializado en validar detecciones mediante razonamiento visual. El sistema analiza la evidencia disponible, determina la validez de cada caso y genera justificaciones contextuales e interpretables.

  • Vision-Language Models
  • Razonamiento visual
  • Explicaciones interpretables
  • Python · Deep Learning
04 Agentes IA

Ecosistema de IA con memoria persistente

Ecosistema de agentes capaz de conservar conocimiento entre interacciones mediante memoria persistente, recuperación contextual y estructuración dinámica de la información. El proyecto evoluciona hacia flujos de decisión avanzados, herramientas externas y capacidades multimodales.

  • Memoria persistente
  • Recuperación contextual
  • Flujos de decisión
  • Python · Agentes IA · Retrieval
05 Deep learning

Eye-tracking basado en inteligencia artificial

Sistema personal de seguimiento ocular mediante deep learning. Incorpora modelos de IA generativa para mejorar la calidad de imagen y técnicas de preprocesamiento y posprocesamiento orientadas a aumentar la precisión, la estabilidad y la fiabilidad del análisis visual.

  • Seguimiento ocular
  • IA generativa
  • Preprocesamiento y posprocesamiento
  • Python · Deep Learning
